# 图片背景去除工具 - MVP 需求文档

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| 产品名称 | Image Background Remover |
| 产品定位 | 轻量级在线图片背景去除工具 |
| 核心价值 | 快速、免注册、即时去除图片背景 |
| 目标用户 | 设计师、电商运营、自媒体创作者、需要快速处理产品图的用户 |

---

## 2. MVP 范围

### 2.1 核心功能（必须）

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 支持拖拽或点击上传，支持 PNG/JPG/WEBP | P0 |
| 背景去除 | 调用 remove.bg API 去除背景 | P0 |
| 结果预览 | 并排展示原图与处理后图片 | P0 |
| 图片下载 | 一键下载处理后的 PNG 图片 | P0 |

### 2.2 扩展功能（暂不做）

- 用户注册/登录
- 历史记录
- 批量处理
- 多种输出格式
- API 开放

---

## 3. 用户流程

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  访问网站    │ ──▶ │  上传图片    │ ──▶ │  处理中...   │ ──▶ │  预览下载    │
└──────────────┘     └──────────────┘     └──────────────┘     └──────────────┘
                           │                    │                    │
                     支持拖拽/点击            加载动画              重新上传
```

---

## 4. 页面设计

### 4.1 首页

**布局**：
- 顶部：标题 + Slogan
- 中间：上传区域（拖拽框）
- 底部：处理结果（双栏对比）+ 下载按钮

**上传区域**：
- 尺寸：最大 1MB（Cloudflare 限制）
- 格式：PNG, JPG, WEBP
- 交互：拖拽上传 + 点击选择

**结果区域**：
- 左侧：原图
- 右侧：去除背景后的图（透明背景）
- 底部：下载按钮 + 重新上传按钮

---

## 5. 技术方案

### 5.1 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | Next.js 14 (App Router) |
| 样式 | Tailwind CSS |
| 后端 | Next.js API Routes |
| AI 服务 | remove.bg API |
| 部署平台 | Cloudflare Pages |
| 图片处理 | 内存流转（不存储）|

### 5.2 架构图

```
┌─────────────┐     ┌─────────────┐     ┌──────────────┐
│  用户浏览器  │ ──▶ │ Cloudflare  │ ──▶ │  remove.bg   │
│             │     │   Pages     │     │    API       │
└─────────────┘     └─────────────┘     └──────────────┘
                           │
                    内存处理，即时返回
                    不存储任何数据
```

### 5.3 环境变量

| 变量名 | 必填 | 说明 |
|--------|------|------|
| REMOVE_BG_API_KEY | 是 | remove.bg API 密钥 |

### 5.4 限制说明

| 限制项 | 限制值 | 原因 |
|--------|--------|------|
| 单图大小 | ≤ 1MB | Cloudflare Workers 请求体限制 |
| 处理时间 | ≤ 30s | API 响应超时 |
| 并发数 | 1 | MVP 暂不做并发控制 |

## 6. 验收标准

### 6.1 功能验收

- [ ] 可通过拖拽上传图片
- [ ] 可通过点击选择图片上传
- [ ] 上传非图片文件时提示错误
- [ ] 上传超过 1MB 图片时提示错误
- [ ] 处理中显示加载动画
- [ ] 处理完成后显示原图与结果图对比
- [ ] 可下载去除背景后的 PNG 图片
- [ ] 可重新上传新图片

### 6.2 性能验收

- [ ] 首页首次加载 < 3s
- [ ] 图片处理时间 < 30s
- [ ] 页面无明显卡顿

### 6.3 部署验收

- [ ] 可通过 `npm run build` 构建成功
- [ ] 可部署到 Cloudflare Pages
- [ ] 环境变量可正常配置
